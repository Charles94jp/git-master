Privacy Policy

Information we collect and how we use it
The Extension use Google Analytics to collect website traffic data that help us improve our service.

When using the Extension, GitHub personal access tokens and/or GitHub OAuth tokens might be needed to access private repositories. These tokens are stored in the browser's storage area. On Safari, these tokens are stored in the browser's local storage. The Extension uses these tokens to authenticate the requests with GitHub (https://github.com). We don’t transmit or store GitHub tokens on our servers. You must take necessary safeguard to ensure your GitHub tokens are secure.

And GitLab、Gitee、Gitea have same behavior.
